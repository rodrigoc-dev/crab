Introdução

Crab é uma linguagem de programação moderna, criada com o objetivo de ser intuitiva, poderosa e eficiente. Seu design foi inspirado no caranguejo, que é ágil, resistente e adaptável — características que queremos transmitir na linguagem. Crab foi projetada para oferecer simplicidade sem comprometer o desempenho, sendo adequada para tudo, desde scripts pequenos até sistemas complexos de alto desempenho.

Se você já possui experiência com outras linguagens de programação, a transição para o Crab será rápida. Caso seja sua primeira linguagem, temos recursos para ajudá-lo a começar do zero.

Instalação

Windows

  1. Baixe o instalador mais recente do Crab na página de Downloads.
  2. Execute o instalador e siga as instruções na tela.
  3. Após a instalação, abra o Prompt de Comando e digite crab --version para verificar se a instalação foi bem-sucedida.

macOS

  1. Abra o Terminal e instale o Crab utilizando o Homebrew:
    2. brew install crab-lang
  2. Verifique se a instalação foi bem-sucedida com o comando crab --version.

Linux

  1. Baixe o arquivo binário do Crab para Linux na página de Downloads.
  2. Extraia o arquivo e mova o binário crab para o diretório /usr/local/bin.
  3. Verifique a instalação com o comando crab --version.

Sintaxe

A sintaxe do Crab foi pensada para ser simples e intuitiva. Abaixo, temos um exemplo básico de código:

fun main() {
    println("Olá, Crab!")
}

Algumas características importantes:

  • O Crab utiliza indentação para definir blocos de código, assim como o Python, garantindo legibilidade.
  • As variáveis podem ser definidas com o tipo inferido automaticamente, ou explicitamente.
  • O comando println() é usado para exibir informações no console.

Tipos de Dados

Crab suporta diversos tipos de dados nativos para facilitar a criação de programas robustos e eficientes. Os principais tipos são:

  • int: Números inteiros, como let x = 42;
  • float: Números com casas decimais, como let y = 3.14;
  • string: Sequências de caracteres, como let nome = "Crab";
  • bool: Valores true ou false, como let isCrab = true;

Você pode criar variáveis de forma simples e clara, e o tipo será inferido automaticamente.

Estruturas de Controle

Crab oferece várias estruturas de controle para permitir decisões e iterações no seu código:

Condicional if / else

let x = 10
if x > 5 {
    println("x é maior que 5")
} else {
    println("x é menor ou igual a 5")
}

Laços de Repetição

O Crab possui os tradicionais laços for e while:

Laço for:
for i in 1..5 {
    println(i)
}
Laço while:
let i = 0
while i < 5 {
    println(i)
    i += 1
}

Funções

Funções são blocos de código que podem ser reutilizados em diversas partes do seu programa. No Crab, você pode definir uma função usando a palavra-chave fun.

Exemplo de definição e uso de função:

fun saudacao(nome: string) {
    println("Olá, " + nome + "!")
}

saudacao("Crab")

Funções podem receber parâmetros e retornar valores. A tipagem é opcional, e o Crab pode inferir tipos automaticamente.

Concorrência

O Crab possui suporte nativo para concorrência, facilitando a execução paralela de tarefas. Isso permite que você aproveite múltiplos núcleos do processador e crie programas mais rápidos e eficientes.

Exemplo de concorrência:

fun main() {
    parallel {
        println("Tarefa 1 executando em paralelo")
    }
    parallel {
        println("Tarefa 2 executando em paralelo")
    }
}

A palavra-chave parallel permite que você execute blocos de código simultaneamente.

Tópicos Avançados

  • Modularização: O Crab permite que você divida seu código em módulos reutilizáveis, aumentando a manutenção e organização do projeto.
  • Tratamento de Erros: O tratamento de exceções no Crab é simples, utilizando a palavra-chave try para capturar erros e catch para lidar com eles.
  • Bibliotecas Externas: O Crab permite o uso de bibliotecas externas para ampliar a funcionalidade da linguagem.

Perguntas Frequentes (FAQ)

1. Quais plataformas o Crab suporta?

O Crab é compatível com Windows, macOS e Linux.

2. Como posso contribuir para o Crab?

A comunidade do Crab é aberta a contribuições! Você pode começar acessando o repositório oficial no GitHub, onde pode reportar problemas, sugerir melhorias e até mesmo enviar pull requests.

3. O Crab é uma linguagem open-source?

Sim! O Crab é totalmente open-source. Seu código-fonte está disponível no GitHub, e incentivamos a participação ativa da comunidade.

4. Quais são os planos futuros para o Crab?

Estamos constantemente aprimorando o Crab, com foco em performance, novos recursos e integração com outras ferramentas de desenvolvimento. Fique de olho nas atualizações no nosso site e repositório oficial.
